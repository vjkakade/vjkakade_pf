## Features

- **Interactive Scrollytelling**: High-resolution image sequence scrubbing via HTML5 Canvas and `framer-motion`, fully optimized using an asynchronous background loading technique to guarantee a 0-second time-to-interactive visual.
- **Retina-Ready Graphics**: Device Pixel Ratio awareness guarantees crisp canvas graphics across high-DPI displays.
- **Modern UI**: Custom dark thematic interface, frosted glassmorphism elements, and fully responsive layouts.
- **Server components & SEO**: Built atop Next.js 14 App Router, heavily utilizing server components and clean metadata structure for top-tier Google indexing.
- **Custom 404 Experience**: Sleek "not found" subdomains complementing the tech-forward ecosystem.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Analytics & Performance Validation

Running a production build executes linting and verifies zero deployment complications:

```bash
npm run build
npm start
```
